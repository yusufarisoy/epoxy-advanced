# Epoxy Advanced
Best practices and advanced usages (EditText & Forms) of Epoxy library written in Modular Architecture.

I used [Rick and Morty API](https://rickandmortyapi.com) in this project.

## Technologies
- Epoxy
- MVVM
- Coroutines
- Flow
- Modular Architecture
- Retrofit2
- ViewBinding
- OkHttp
- Gson
- Glide
